# React + Express + MySql을 이용한 Dwitter 프로젝트

[프로젝트 바로가기](https://dwitters.netlify.app/)

> client 폴더 코드에 직접 작성한 node.js의 Express를 적용해 만든 프로젝트

> 프론트 (client)
> - React

>백엔드 기술 스택 
> - express
> - mysql
> - Sequelize (ORM)

> 공통 
> - socket.io

> 배포
> - 프론트 - netlify로 client 배포 
> - 백 - heroku로 server 배포
